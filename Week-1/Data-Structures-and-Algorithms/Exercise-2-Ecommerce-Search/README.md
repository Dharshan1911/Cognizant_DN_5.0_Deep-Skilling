# Exercise 2 - E-commerce Platform Search

## Objective

Implement Linear Search and Binary Search to search products by Product ID.

## Concepts Used

- Arrays
- Linear Search
- Binary Search
- Sorting
- Time Complexity

## Time Complexity

| Algorithm | Complexity |
|-----------|------------|
| Linear Search | O(n) |
| Binary Search | O(log n) |

## Output

```
Linear Search
Product ID: 103, Name: Phone, Category: Electronics

Binary Search
Product ID: 103, Name: Phone, Category: Electronics
```
