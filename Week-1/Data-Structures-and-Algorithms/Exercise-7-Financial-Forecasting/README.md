# Exercise 7 - Financial Forecasting

## Objective

Predict future financial value using recursion.

## Formula

Future Value = Present Value × (1 + Growth Rate)^Years

## Concepts Used

- Recursion
- Mathematical Modeling
- Time Complexity

## Complexity

Time Complexity: O(n)

Space Complexity: O(n)

## Expected Output

```
Current Value : ₹10000.00
Growth Rate   : 8%
Years         : 5
Future Value  : ₹14693.28
```
