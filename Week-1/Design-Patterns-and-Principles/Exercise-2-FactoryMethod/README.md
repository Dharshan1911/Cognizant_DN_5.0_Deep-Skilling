# Exercise 2 - Factory Method Pattern

## Objective

Implement the Factory Method Design Pattern using Java.

## Description

This project demonstrates the Factory Method Pattern by creating different types of documents (Word, PDF, Excel) through separate factory classes without exposing object creation logic to the client.

## Files

- Document.java
- WordDocument.java
- PdfDocument.java
- ExcelDocument.java
- DocumentFactory.java
- WordFactory.java
- PdfFactory.java
- ExcelFactory.java
- FactoryMethodExample.java

## Expected Output

```
Opening Microsoft Word Document...
Opening PDF Document...
Opening Excel Spreadsheet...
```
