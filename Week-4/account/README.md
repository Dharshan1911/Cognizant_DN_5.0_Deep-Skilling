# Account Microservice

## Overview
This project is part of Cognizant Digital Nurture 4.0 - Week 4.

The Account Microservice is developed using Spring Boot and exposes REST APIs to retrieve customer account information.

## Technologies Used

- Java 21
- Spring Boot
- Spring Web
- Maven
- REST API

## Project Structure

```
account/
├── src/
├── pom.xml
├── README.md
└── outputs/
```

## How to Run

1. Open the project in IntelliJ IDEA.
2. Ensure Java 21 SDK is selected.
3. Wait for Maven dependencies to download.
4. Run `AccountApplication`.
5. Open the browser and test the endpoint.

Example:

```
http://localhost:8046/accounts
```

## Output

Successful execution returns account details through REST API.

Screenshots are available inside the `outputs` folder.
